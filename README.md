# 플라월드(Flowerworld)

![image](flowerworld.png)

주어진 암호문을 따라 꽃밭 미로를 빠져나가자!

* PC / 1인용 퍼즐 게임 / Unity
* 개발기간 : 2017 2학기 + 2018 겨울합숙 개발
* 개발팀 : 김승현 김영찬 백성빈 유선주 정유철 + 외주, 동아리원 및 외부 테스터 다수 협력
